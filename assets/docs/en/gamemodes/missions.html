The missions system allows the player to complete a variety of activities, like driving from point A to B, setting the fastest time on a set course, getting a high drifting score, etc.

The main difference with the scenario system is that missions are accessed directly from markers in sandbox mode, instead of being in a separate menu.

Creating your own missions is relatively straight forward, and is done using the world editor.

Once that is done, the next step is to chose the type of mission you want to create.

  • Premade mission types include a variety of easily configurable gameplay loops.
  • Flowgraph missions are done using the flowgraph system, which is a lot more configurable, but also more complex than building missions with pre-made types.
