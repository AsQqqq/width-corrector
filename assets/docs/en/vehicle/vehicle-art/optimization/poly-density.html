Explanation

Polycount is not really relevant, but more so the poly density.

The reason for this is because the GPU only cares about how small the polygons are, because it needs to perform many more calculations if the polygon is under a certain size.

Checking Problem Areas

You can use the Poly Density Debug to identify issues.

Where to find this tool?

  1. Open World Editor F11
  2. Go to View → Visualization Settings
  3. Search for Triangle size

What does it mean?

  • Blue polygon costs what ever a polygon costs in-game
  • Green means costs 200% more of blue
  • Yellow means costs 300% more of blue
  • Red means costs 400% more of blue

How much red is acceptable?
Not sure, because this depends on min spec testing.
