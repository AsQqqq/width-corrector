Overview

This Workflow requires Blender and Substance Painter - nothing else. The result will be a baked glowmap which gives the user an impression of bounced light.

The focus here is on 3D Lights - depending on the use case, Flat 2D light may require a slightly different technique.

The example here is based on the Bastion Project.

You can find all the files used here in vehicle_artwork/bastion/source/lights_updated

You will end up with the best result when you got a normalmap set up as well, because the light can interact with it to produce some nice details.

Getting Started in Blender

First you grab all the meshes making use of the material which gets linked to the glowmap and place them in a new collection.

You got the material assigned, in this case its bastion_lights. The said material then needs the normalmap applied in the shader editor, which you can easily do through a simple node setup, containing an Image Texture and Normal Map node.

Keep in mind to have the Image Texture Color Space set to Non-Color, since you are using a normal map.

Play around with the roughness values in the BSDF as well. That way you can fine tune the way the light gets reflected on the surface, and use a almost black Base Color - this is important for the mask work in substance painter, as we are basically only producing masks which we are going to work with later in Blender right now.

Setting up the glow

The next step is where the light comes in. Grab a point light as your light source and basically place it where the bulb would be.

Once the lights have been placed, you can see that they do interact with the applied normalmap. Keep in mind that the light source should be white, as we aim to produce a mask here.

What’s important for the baking is that you need to disable the camera icon for the other lights that are irrelevant for the particular setup you want to bake - the camera defines whether something gets rendered or not. In this case we want to bake the front drl collection only.

Baking

Next, let’s proceed with the baking, for this you need to use the Cycles Render Engine.

Once that is selected you’ll get a bunch of tabs you can open, one of them being Bake - that’s the one we want. Here you set up a Combined bake like shown, and adjust the margin to your liking.

Happy with your setup? Cool - now, select the mesh you want the glowmap to be baked on. In this case, we want to bake the DRL’s for the bastion_headlight_R mesh. We select it with left-click and hit the Bake button.

Once the baking is done you should end up with an image similar to this:

This result can now be used to be imported into Substance Painter as bitmap mask.

Continue in Substance Painter

In your painter project you create a fill layer with emission, then add the image you just produced as bitmap mask.

Don’t forget to add an Emissive channel to your project in the Texture Set Settings

Once that is in, I recommend playing with the mask’s levels a bit since we didn’t use a full black for the baking. Just move the black tone a bit so that we end up with a pitch black for the black areas on the mask.

Set the emissive to white, and you got your glowmap working. Of course you can do that with any other color as well - the Painter workflow is super easy to tweak on the go.

Keep in mind to also set up an emissive output when you export the material!

Baking Emissive Maps to Glass

To get this type of texture that can be used as bitmap mask, you just need to make a few adjustments to the materials you’re working with. The rest of the workflow is the same.

Basically, you leave the point lights where they are, but make the glass covering the light housing visible.

Its important to have both meshes enabled for rendering (including the point lights of course) so the light bounding inside the housing also affects the way the illumination is being baked onto the glass.

Also important is to set the glass base color to a very dark value too - like this:

The rest of the material setup can be adjusted to your liking - but keep in mind to play around with transmission and transmission roughness to get a result that suits your vehicle
