The information section contains data that is to be shown to the user in the User Interface (UI).

The section is a simple dictionary with a few simple optional arguments:

string
type
Name of the author
Please put your name into this field. Multiple authors can be separated by comma.
BeamNG is reserved for official vehicles and should not be used.
string
type
Visual name of the part

Data in jbeam on the left, how it shows in-game on the right:

Cannot start with the $ sign as that is reserved for expressions. You can put a space at the start to bypass this and it will display as it should, for example:

"name":" $36M Pallet",
string
type
Monetary value of the part in US dollars
Used in career mode to buy and sell parts for realistic values of money.
boolean
type
false
default
Whether to hide the part in the parts selector
When set to true, the part will only show up in parts selector when “Include auxiliary debug content” option is checked.

Simple Example 

{
  "information":{
    "authors":"Max Mueller",
    "name":"Front Spindles",
    "value":100,
  },
}
