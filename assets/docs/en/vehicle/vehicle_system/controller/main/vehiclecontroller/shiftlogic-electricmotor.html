Summary

This controller handles the shifting and regeneration functions of drive trains using an electric motor.

It is an extension of the vehicle controller . As it is not a type of transmission, it requires using the “shiftLogicName” property with “electricMotor” as shown in the example at the bottom of the page.

Example Usage

A vehicle controller section with parameters specific to this sub controller.

    "vehicleController": {
        "shiftLogicName":"electricMotor",
        "motorNames": ["rearMotor", "frontMotor"],
        "topSpeedLimitReverse": 15,
        "defaultRegen": 0.5
        "brakeRegenCoef": 1
    },
