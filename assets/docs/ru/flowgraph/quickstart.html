Обзор

На этой странице представлено краткое руководство по запуску Flowgraph и введение в основные элементы управления редактором.
Это хорошая отправная точка для первых шагов в Flowgraph, прежде чем углубляться в изучение всех его возможностей.


Открытие проекта

Сначала рассмотрим, как открыть один из примеров проектов для Flowgraph.
Находясь в главном меню BeamNG.drive, просто нажмите F11, чтобы открыть WorldEditor , а затем выберите Window -> Gameplay -> Flowgraph Editor в заголовке меню.


Сейчас вы видите окно «Welcome» Flowgraph, поскольку ни один проект ещё не открыт.
Здесь находится раздел «Basic Examples», в котором можно выбрать проект «EditorBasics» для его открытия.


Основные элементы управления

Навигация по графу

После открытия проекта «EditorBasics» вы увидите разнообразные элементы интерфейса.
Наиболее важным является большое окно в центре — это «Main»-окно Flowgraph, отображающее текущий открытый граф. Поскольку именно здесь вы будете работать с флоуграфом, важно уметь удобно перемещаться по нему.
Чтобы переместить вид в графе, просто удерживайте Right Click и перетаскивайте камеру.


Проект Flowgraph может состоять из нескольких флоуграфов и стейтграфов. Стейтграфы — более продвинутая тема, подробнее о них можно прочитать на странице Elements .
Чтобы переключиться на другой граф в проекте, достаточно выбрать соответствующую вкладку в верхней части главного окна.


Соединение нод

Рассмотрим сам флоуграф.
Флоуграф состоит из нод с пинами, которые соединены между собой связями (links), что и оживляет граф. Подробнее об этих базовых элементах можно прочитать на странице Elements .
Соединение нод — одна из самых базовых функций Flowgraph и может выполняться различными способами. Простейший из них — нажать Left Click на пин и перетащить связь к пину другой ноды.
Если соединить два пина не удаётся, скорее всего, один из них ожидает другой тип данных. Также учтите, что соединять можно только входной пин с выходным.


Создание нод

При работе с Flowgraph вам, конечно, понадобится добавлять новые ноды в граф.
Как и со связями, это можно сделать несколькими способами. Самый простой — нажать Right Click на свободном месте флоуграфа, что вызовет меню «Node Creation».
Здесь можно выбрать ноды из различных категорий и добавить их в граф.


Скрытие связей

Для улучшения читаемости флоуграфа вы можете скрывать и отображать связи. Для этого выберите связь и нажмите H — это переключит видимость связи.
Скрытую связь можно выбрать, кликнув на одну из соединённых нод, а затем нажав на появившийся «призрак» связи.


Запуск проекта

Конечно, вы захотите увидеть флоуграф в действии и запустить проект.
Для этого просто нажмите на панели инструментов окна «Main», а для остановки нажмите .

Пока флоуграф активен, из ноды «on Update» будет идти поток, распространяющийся по всему графу и активирующий все ноды, до которых он добирается. Подробнее о потоке можно прочитать на странице Flow .
Flowgraph предоставляет широкий набор нод с различными функциями, давая вам инструменты для создания сложных сценариев.

Это краткое руководство дало лишь общее представление о навигации в редакторе Flowgraph.
