Общее

«FG Monitor» — это полезный инструмент для отладки вашего флоуграфа во время выполнения.


Он предоставляет 3 основные функции:

  • Панель инструментов с кнопками для:

    • скрытия монитора и открытия WorldEditor.
    • остановки проекта.

  • Обзор активных состояний в вашем флоуграфе.

  • Обзор всех зафиксированных событий, аналогичный окну «Event Log» во Flowgraph.

Включение и отключение

Монитор можно легко переключить, нажав флажок Use Monitor на панели инструментов «Main Window» в редакторе Flowgraph.
