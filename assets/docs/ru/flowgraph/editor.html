Обзор

Редактор Flowgraph — это ваш инструмент для создания, изменения и отладки проектов Flowgraph.

Редактор Flowgraph можно открыть, нажав F11 для запуска WorldEditor, а затем кликнув Window => Gameplay => Flowgraph Editor.
Сам редактор состоит из набора окон с различными функциями. Вы можете открывать или скрывать эти окна с помощью выпадающего меню Windows в главной панели инструментов редактора.
Чтобы узнать подробнее о различных окнах редактора Flowgraph и их функциях, ознакомьтесь со страницей Windows.


Проекты Flowgraph

Проекты Flowgraph хранятся в обычных файлах JSON с расширением ".flow.json". Проект Flowgraph существует независимо от уровня или сценария, поэтому файл может быть сохранён в любой директории.


Создание проектов

Существует несколько способов создать проект Flowgraph:

  • Выбрать File => New Project в редакторе Flowgraph.
  • Нажать New Project в окне "Welcome" редактора Flowgraph.
  • Скопировать существующий проект.

Открытие проектов

Существует несколько способов открыть проект Flowgraph:

  • Выбрать File => Load Project… и указать файл проекта.
  • Выбрать File => Recent Files… и затем выбрать один из недавних проектов.
  • Выбрать File => Restore… для доступа к предыдущей версии проекта.
  • Нажать Open Project в окне "Welcome" редактора Flowgraph.
  • Выбрать примеры из окна "Welcome" или "Examples" редактора Flowgraph.

Можно открывать несколько проектов Flowgraph одновременно. Управлять открытыми проектами можно в окне "Task Manager" редактора Flowgraph.


Сохранение проектов

Существует несколько способов сохранить проект Flowgraph:

  • Выбрать File => Save или File => Save as в редакторе Flowgraph.
  • Выбрать Save или Save as в окне "Project Settings".
  • Нажать сочетание клавиш Ctrl+S, находясь в редакторе Flowgraph.

Запуск & остановка проектов

Существует несколько способов запустить проект Flowgraph:

  • Нажать на панели инструментов окна "Main" в редакторе Flowgraph.
  • Нажать на проекте в окне "Task Manager" в редакторе Flowgraph.
  • Запустить сценарий, с которым связан файл Flowgraph, а затем открыть WorldEditor.

Остановить запущенный проект Flowgraph можно следующими способами:

  • Нажать на панели инструментов окна "Main" в редакторе Flowgraph.
  • Нажать на проекте в окне "Task Manager" в редакторе Flowgraph.
  • Нажать в окне "FG Monitor", если оно используется.

Примеры

Доступен широкий набор примеров проектов, с помощью которых можно ознакомиться с различными возможностями работы с Flowgraph. Эти проекты можно найти в окне "Welcome" или в окне "Examples".

Если вы только начинаете работу с Flowgraph, рекомендуем начать с проектов для начинающих, чтобы освоить основные концепции. Также вы можете ознакомиться с нашим руководством Introduction to Flowgraph для более подробного изучения идей Flowgraph.
Кроме того, вы всегда можете открыть существующие сценарии и нажать F11, чтобы проверить, есть ли у них связанный файл Flowgraph.

Как и весь Flowgraph, эта часть находится в активной разработке, и в будущем будут добавлены новые примеры проектов.
