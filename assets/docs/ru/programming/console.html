Вы можете запустить Bin64/console.x64.exe в качестве удобной небольшой (интерактивной) lua-консоли.

console.x64.exe cmd <lua string to execute>
console.x64.exe <lua file to execute>

Например, для выполнения lua-строки:

console.x64.exe cmd "print(1+1)"

Например, для выполнения lua-файла:

console.x64.exe test.lua

Для интерактивной оболочки запустите без аргументов:

console.x64.exe

Ниже приведена краткая документация по её весьма простому API

BeamNG console Lua API Documentation

Классы

HighPerfTimer

Высокопроизводительный таймер для точных измерений времени.

local timer = HighPerfTimer()

Методы

  • reset() - Сбрасывает таймер в ноль
  • elapsed() - Возвращает прошедшее время в секундах с момента создания или последнего сброса
  • stopAndReset() - Возвращает прошедшее время и сбрасывает таймер
  • stop() - Псевдоним для elapsed() (устарело)

BeamEngine

Основной интерфейс движка.

Методы

  • update(dtReal, dtSim) - Обновляет симуляцию движка
  • spawnObject2(...) - Создаёт объект в симуляции
  • deleteAllObjects() - Удаляет все объекты из симуляции
  • setDynamicCollisionEnabled(enabled) - Включает/отключает динамическое столкновение
  • instabilityDetected() - Проверяет, была ли обнаружена нестабильность в симуляции

BeamObject

Представляет физический объект в симуляции.

Методы

  • getNodeCount() - Возвращает количество нод в объекте
  • getBeamCount() - Возвращает количество балок в объекте

VFS

Интерфейс виртуальной файловой системы.

Доступ к VFS осуществляется через глобальную переменную FS

Методы

  • fileSize(filename) - Возвращает размер файла в байтах
  • fileExists(filename) - Проверяет, существует ли файл
  • directoryExists(directory) - Проверяет, существует ли директория
  • getFileRealPath(virtualPath) - Получает реальный путь файловой системы из виртуального пути
  • getGamePath() - Получает корневой путь игры
  • openFile(filename, mode) - Открывает файл
  • copyFile(src, dst) - Копирует файл
  • directoryCreate(name, recursive) - Создаёт директорию
  • findFiles(path, pattern, recursiveLevels, multiMatch, includeDirs) - Находит файлы, соответствующие шаблону
  • hashFile(filename) - Получает хеш файла
  • hashFileSHA1(filename) - Получает SHA1-хеш файла
  • stat(filename) - Получает статистику файла
  • removeFile(filename) - Удаляет файл
  • renameFile(oldName, newName) - Переименовывает файл
  • mount(source, target, priority) - Монтирует директорию или архив
  • mountList(mountList) - Монтирует несколько путей
  • unmount(path) - Размонтирует путь
  • isMounted(path) - Проверяет, смонтирован ли путь

Глобальные функции

  • initBeamEngine(steps) - Инициализирует движок BeamNG
  • destroyBeamEngine() - Завершает работу движка BeamNG
  • reloadLua() - Перезагружает среду Lua
  • log(level, origin, msg) - Записывает сообщение в лог
    • level: Уровень лога (I=Info, W=Warning, E=Error, D=Debug)
    • origin: Источник лога
    • msg: Сообщение для записи
  • updateLuaCore() - Обновляет основную систему Lua
  • profilerPushEvent(name) - Начинает профилирование события
  • profilerPopEvent() - Завершает профилирование текущего события
  • setPowerPlanMaxPerformance() - Устанавливает план электропитания системы на максимальную производительность
  • restorePowerPlan() - Восстанавливает исходный план электропитания

Глобальные переменные

  • hw - Таблица с информацией об аппаратном обеспечении
    • hw.bits - Архитектура CPU (32 или 64 бит)
  • args - Аргументы командной строки
  • beamng_platform - Текущая платформа ("windows", "linux" и т.д.)
