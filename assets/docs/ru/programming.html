Как моддер, рано или поздно вы можете захотеть создать расширенную функциональность для своего мода. Эта функциональность может выходить за рамки того, что предлагает программное обеспечение BeamNG по умолчанию, и потребует написания собственного программного кода. Как правило, для игрового процесса используется код на Lua, а для пользовательского интерфейса — HTML/JS.

Цель здесь — дать общее руководство, объяснив общие концепции и архитектурные решения, специфичные для BeamNG.

Это означает, что:

  • Подробная документация по API выходит за рамки данной документации.
  • Обучающие материалы по программированию, веб-разработке и т.д. также выходят за рамки данной документации. В Интернете есть множество более качественных источников знаний, и мы советуем начать именно с них, если вы ещё не являетесь программистом.

Для любой специфичной для BeamNG информации, которую вы не можете найти в этой документации, например подробного использования API и т.д., лучше всего:

  • а) Изучить существующий код и использовать его в качестве приблизительного ориентира
  • б) Обратиться за помощью по программированию, специфичному для beamng, на наших форумах по программированию .
