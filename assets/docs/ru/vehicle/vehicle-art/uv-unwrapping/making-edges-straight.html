Рабочий процесс UV для кузовных панелей и декоративных элементов с фасками

Этот документ описывает специальный рабочий процесс создания UV-развёрток кузовных панелей и декоративных элементов с скошенными границами (bevels).

Чтобы добиться наилучших результатов и избежать артефактов сглаживания (anti-aliasing) на фасках, UV-развёртки должны создаваться в соответствии со следующими правилами.

Правила и рекомендации

  • UV-границы панелей, которые можно выровнять, должны быть выровнены
  • Избегайте лишних UV-швов на поверхности панели
    • Если UV-шов необходим, размещайте его по наиболее скрытому пути (поскольку при рендеринге в игре UV-шов может проявляться как видимый жёсткий край)
  • Все кузовные панели должны создаваться с одинаковой плотностью пикселей (это обеспечивает единую ширину фаски по всем границам)
  • Раскладка UV должна быть оптимизирована
    • Сжимайте прямые участки UV-оболочки, которые не влияют на фаски
    • Совмещайте похожие элементы, например UV с прямоугольными формами, чтобы максимально эффективно использовать пространство текстуры
