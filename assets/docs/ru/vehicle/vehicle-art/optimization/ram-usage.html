Проверка использования RAM

Для выявления проблем можно воспользоваться следующим встроенным инструментом:

  1. Запустите игру в режиме Direct X (этот инструмент пока не работает в Vulkan)
  2. Откройте График производительности ( Ctrl+Shift+F) → GPU Memory Profiling
  3. Убедитесь, что включён параметр Update Data на вкладке General
  4. Выполните поиск по запросу vehicle:Texture

Для получения более детальной информации:

  1. Перейдите на вкладку GPU Memory Profiling
  2. Откройте раздел GFX Texture Objects внутри неё
  3. Разверните меню Filter
  4. Выберите vehicle и снимите выделение со всего остального в списке, чтобы увидеть дополнительные данные
Дополнительные советы см. в разделе Оптимизация материалов и мешей .

Типичные проблемы

Проблема Очень высокое потребление видеопамяти (GPU RAM), которое необходимо снизить — целевое значение ниже 300 МБ.
Решение Уменьшите размеры текстур до нужного уровня; можно также использовать новое творческое тайлинг-разбиение текстур для сохранения плотности текстур.

Проблема Иногда мы добавляем детали, которые используют лишь 1% огромного набора текстур.
Решение Например, до недавних изменений добавление антенны к транспортному средству загружало в RAM весь набор текстур внедорожных деталей, хотя автомобилю требовалось всего несколько пикселей из него.

Другие типичные проблемы:

  • Карты AO нередко имеют излишне высокое разрешение. Поскольку они мягкие и лишены чётких деталей, их можно значительно уменьшить без видимых потерь качества.
  • Карты лакового покрытия (cc.data) как правило малозаметны или даже дублируют карты непрозрачности, поэтому высокое разрешение им не нужно или вовсе излишне.
  • Металличность и карты шероховатости обычно просты по содержанию, но при этом хранятся в очень высоком разрешении. Их часто можно уменьшить — тем более что более детальные текстуры, например карты нормалей, способны скрыть артефакты низкого разрешения. Пример ниже — карта металличности, которую можно существенно уменьшить, поскольку в ней нет никаких деталей.
Всегда учитывайте взаимодействие разных карт. Детализированная карта нормалей 4K может скрыть недостатки карт шероховатости или металличности с более низким разрешением.
