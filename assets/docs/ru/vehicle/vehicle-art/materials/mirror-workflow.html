Именование материалов для мешей

  • mirror_CE - Вогнутое (зеркало со стороны водителя)
  • mirror_CX - Выпуклое (зеркало со стороны пассажира)
  • mirror_F - Плоское (центральное зеркало)

Убедитесь, что UV-развёртка зеркал полностью покрывает всю область карты, чтобы избежать проблем с разрешением!

После того как вы закончите перемещение зеркал в нужное место в 3D, камера зеркала может оказаться не там, где нужно — в этом случае её можно дополнительно скорректировать с помощью функции, описанной ниже.

  • Шаг 3: Откройте Notepad++ и нажмите Ctrl+F, переключитесь на вкладку «Find in Files» в появившемся окне, затем введите имя этого зеркала в строку поиска с фильтром *.jbeam и укажите в качестве директории папку редактируемого транспортного средства (также можно указать всю папку vehicles, однако поиск займёт больше времени)
  • Шаг 4: Нажмите на один из найденных результатов поиска
  • Шаг 5: Найдите секцию "mirrors"
  • Шаг 6: Перенесите параметры из режима отладки в JBeam — refBaseTranslation соответствует ползункам положения по XYZ, а baseRotationGlobal указан здесь (его значение получается путём настройки ползунков XYZ с последующим вводом полученного параметра поворота в JBeam):
  • Шаг 7: Сохраните файл JBeam
  • Шаг 8: Перезагрузите транспортное средство нажатием Ctrl+R
  • Шаг 9: Готово!
