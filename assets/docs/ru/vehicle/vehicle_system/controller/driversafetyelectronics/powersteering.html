Обзор

Контроллер усилителя руля используется для имитации воздействия гидроусилителя руля на усилия, передаваемые на рулевое колесо.

Он влияет только на интенсивность обратной связи по усилию.

Контроллер настроен таким образом, чтобы обеспечивать обратную связь, изменяющуюся в зависимости от скорости.

Информация о файле

Имя файла powerSteering.lua
Путь /lua/vehicle/controller/powerSteering.lua
Тип Auxiliary
Тип использования Single

Свойства JBeam

number
type
1
default
Уровень усиления рулевого управления на высокой скорости.
Меньшие значения означают меньшую силу, передаваемую на руль. Значение 1 означает, что рулевое управление практически не имеет усиления, тогда как 0 означает полное отсутствие передачи усилий обратно.
number
type
1
default
Уровень усиления рулевого управления на низкой скорости.
Меньшие значения означают меньшую силу, передаваемую на руль. Значение 1 означает, что рулевое управление практически не имеет усиления, тогда как 0 означает полное отсутствие передачи усилий обратно.
number
type
0
default
Скорость транспортного средства (м/с), при которой используется значение strengthSlow.
Ниже этой скорости используется значение strengthSlow. Между этой скоростью и speedFast контроллер будет интерполировать значение между strengthFast и strengthSlow.
number
type
0
default
Скорость транспортного средства (м/с), при которой используется значение strengthFast.
Ниже этой скорости используется значение speedFast. Между этой скоростью и speedSlow контроллер будет интерполировать значение между strengthFast и strengthSlow.
number
type
-0.2
default
Постоянный модификатор уровня усиления.

Отрицательные значения означают, что рулевое управление будет вычитать определённое количество передаваемой силы после завершения расчётов strengthFast/Slow.

Игра всегда ограничивает значение минимумом 0.

Пример использования 

"controller": [
    ["fileName"],
    ["powerSteering", {"strengthFast":1, "strengthSlow": 0.10, "speedSlow": 2, "speedFast": 20, "constantOffset":-0.05}],
],
