Краткое описание

Этот контроллер позволяет добавить анимированный экран в комбинацию приборов транспортного средства, отображающий полезную информацию.

Информация о файле

Имя файла etkGauges.lua
Путь /lua/vehicle/controller/etkGauges.lua
Тип Вспомогательный
Тип использования Множественный (не используется)
