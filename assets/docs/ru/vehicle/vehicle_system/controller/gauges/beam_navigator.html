Обзор

Простой контроллер для добавления навигационного экрана внутри транспортного средства. Для отображения на экране требуется правильно настроенный материал.

Сведения о файле

Имя файла beamNavigator.lua
Путь /lua/vehicle/controller/beamNavigator.lua
Тип Auxiliary
Тип использования Multiple

Свойства JBeam

string
type
@screen_gps
default
Название материала, используемого для экрана навигатора.
string
type
local://local/vehicles/common/navi_screen.html
default
HTML-файл, отображаемый на экране навигатора.
number
type
256
default
Ширина текстуры (пикс.).
number
type

Высота текстуры (пикс.).
number
type

default

Пример использования 

"controller": [
  ["fileName"],
  ["beamNavigator"]
],
"beamNavigator": {
  "screenMaterialName":"@screen2_gps",
  "textureWidth":512,
  "textureFPS": 20,
}
