Описание

Этот контроллер позволяет управлять одним или несколькими индикаторами переключения передачи, которые сигнализируют о необходимости переключиться на основании оборотов двигателя.

Он позволяет настроить несколько ступеней, которые будут последовательно загораться по мере приближения двигателя к максимальным оборотам.

Пример использования

Индикатор переключения передачи с одной ступенью.

"controller": [
    ["fileName"],
    ["shiftLights"],
],
"shiftLights": {
    "engineName":"mainEngine",
    "maxEngineRPMOffset": 400,
    "rpmRange": 1000,
    "outputElectrics": ["led"],
    "flashingOutputElectrics": ["led"],
},

Индикатор переключения передачи с 7 ступенями, причём последние две мигают при достижении ограничителя оборотов.

"controller": [
    ["fileName"],
    ["shiftLights"],
],
"shiftLights": {
    "engineName":"mainEngine",
    "maxEngineRPMOffset": 400,
    "rpmRange": 3000,
    "keepLEDsActive": true,
    "hasStallWarning": true,
    "outputElectrics": ["led0", "led1", "led2", "led3", "led4", "led5", "led6", "led7"],
    "flashingOutputElectrics": ["led6", "led7"],
    "flashingAlternateOutputElectrics": [],
},
