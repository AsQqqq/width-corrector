Краткое описание

Контроллер впрыска закиси азота используется для управления системой закиси азота .

Сведения о файле

Имя файла nitrousOxideInjection.lua
Путь /lua/vehicle/controller/nitrousOxideInjection.lua
Тип Auxiliary
Режим использования Multiple

JBeam Properties

string
type
Имя данного экземпляра контроллера закиси азота.
Допускается использование нескольких контроллеров с разными именами, однако это не рекомендуется. Как правило, несколько баллонов привязываются к одному контроллеру.
string
type
nitrousOxideArm
default
Имя электрического сигнала, который активируется при взводе системы.
Должно совпадать со значением, заданным в элементе трансмиссии закиси азота.
string
type
nitrousOxideOverride
default
Имя электрического сигнала, который активируется при включении режима принудительной подачи закиси азота.
Должно совпадать со значением, заданным в элементе трансмиссии закиси азота.
