Краткое описание

Трансбрейк работает путём блокировки коробки передач при включённой передней передаче, позволяя нагружать двигатель без движения автомобиля вперёд.

Это часто используется в сочетании с системой launch control для набора давления наддува и предварительной нагрузки гидротрансформатора на более мощных автомобилях для улучшенного старта.

При наличии трансбрейка для его активации по умолчанию используется комбинация клавиш Shift + Ctrl + Alt + Space.
