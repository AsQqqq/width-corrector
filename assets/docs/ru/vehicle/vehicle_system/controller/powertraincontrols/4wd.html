Краткое описание

Контроллер полного привода (4WD) позволяет подключать и отключать карданный вал, а также управлять раздаточной коробкой. Для корректной работы в конфигурации необходимо указать правильные имена устройств трансмиссии (shaft и rangeBox).

При необходимости можно использовать несколько контроллеров 4WD одновременно — каждый из них будет управлять отдельным набором устройств.
