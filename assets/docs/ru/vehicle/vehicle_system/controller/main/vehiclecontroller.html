Краткое описание

Контроллер транспортного средства является главным контроллером для любого приводного транспортного средства в beamNG. Он управляет органами управления силовой передачей и взаимодействует с различными контроллерами логики переключения передач для различных типов трансмиссий.

Либо этот контроллер, либо контроллер-заглушка должен быть вызван на любом транспортном средстве или реквизите для его правильной работы.

Он является основным источником данных electrics о двигателе и информации о трансмиссии.

Пример использования 

    "controller": [
        ["fileName"],
        ["vehicleController", {}],
    ],
    
    "vehicleController": {
        "calculateOptimalLoadShiftPoints": true,
        "transmissionGearChangeDelay":2,
        "gearboxDecisionSmoothingUp":2,
        "gearboxDecisionSmoothingDown":2,
        "aggressionSmoothingUp":1,
        "aggressionSmoothingDown":0.5,
        "aggressionHoldOffThrottleDelay":1.75,
        "lowShiftDownRPM":1000,
        "lowShiftUpRPM":1800,
    },

Свойства vehicleController транспортного средства обычно распределены между различными компонентами двигателя и трансмиссии.

Компоненты, снижающие крутящий момент на низких оборотах, могут повышать значения clutchLaunch, тогда как компоненты, увеличивающие максимальные обороты двигателя, повышают значения highShiftRPM.
