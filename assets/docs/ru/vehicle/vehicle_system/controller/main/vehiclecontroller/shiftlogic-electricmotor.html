Краткое описание

Этот контроллер управляет переключением передач и функциями рекуперации трансмиссий, использующих электродвигатель.

Он является расширением контроллера транспортного средства . Поскольку это не тип трансмиссии, требуется использование свойства "shiftLogicName" со значением "electricMotor", как показано в примере в нижней части страницы.

Пример использования

Раздел контроллера транспортного средства с параметрами, специфичными для данного подконтроллера.

    "vehicleController": {
        "shiftLogicName":"electricMotor",
        "motorNames": ["rearMotor", "frontMotor"],
        "topSpeedLimitReverse": 15,
        "defaultRegen": 0.5
        "brakeRegenCoef": 1
    },
