Краткое описание

Контроллер логики переключения автоматической коробки передач управляет автоматическим переключением передач трансмиссии, различными режимами движения и блокировкой гидротрансформатора.

Является расширением контроллера автомобиля и автоматически вызывается, если транспортное средство оснащено автоматической трансмиссией .

Свойства JBeam

string
тип
PRNDS21M
по умолчанию
Доступные режимы переключения передач.

Трансмиссия только с ручными режимами не будет переключать передачи автоматически в режиме реалистичного переключения.

Это можно использовать для имитации автоматических трансмиссий с ручным управлением клапанами, как на некоторых гоночных автомобилях.

Пример использования

Секция контроллера автомобиля с параметрами, специфичными для данного подконтроллера, определённая внутри jbeam трансмиссии.

"vehicleController": {
    "automaticModes":"PRND21",
    "useSmartAggressionCalculation":false,
    "calculateOptimalLoadShiftPoints": true,
    "transmissionGearChangeDelay":2,
    "gearboxDecisionSmoothingUp":0.5,
    "gearboxDecisionSmoothingDown":0.5,
    "lowShiftDownRPM":1100,
    "lowShiftUpRPM":1900,
    //"highShiftDownRPM":3100,
},

Секция контроллера автомобиля для блокирующегося гидротрансформатора.

"vehicleController": {
    "torqueConverterLockupRPM":1100,
    "torqueConverterLockupMinGear":3,
},
