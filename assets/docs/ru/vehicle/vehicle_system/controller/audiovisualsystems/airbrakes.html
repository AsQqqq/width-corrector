Описание

Контроллер воздушных тормозов используется для управления звуковыми событиями на транспортных средствах с пневматическими тормозами, таких как автобусы и грузовые седельные тягачи.

При отпускании тормозов воспроизводится звук, интенсивность которого регулируется в зависимости от степени их отпускания.
