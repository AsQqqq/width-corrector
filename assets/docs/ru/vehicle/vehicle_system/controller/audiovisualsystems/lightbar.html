Краткое описание

Контроллер Lightbar позволяет реализовывать пользовательские паттерны мигания световой панели. Также поддерживается определение нескольких режимов мигания, между которыми можно переключаться.

Информация о файле

Имя файла lightbar.lua
Путь /lua/vehicle/controller/lightbar.lua
Тип Вспомогательный
Тип использования Множественный

JBeam Properties

array
type
Массив режимов световой панели, между которыми можно переключаться.
Подробнее о доступных параметрах см. в разделе режим световой панели.
number
type
1
default
Режим по умолчанию, с которого начинает работу световая панель.

lightbarMode

string
type
Название режима световой панели.
array
type
Массив данных паттерна для данного режима.
Подробнее о доступных параметрах см. в разделе паттерн режима освещения.

lightbarModePattern

string
type
Название переменной electrics для управления.
number
type
Значение, которое устанавливается для electric.
number
type
Длительность установки значения перед переключением на следующее значение для данного electric.
