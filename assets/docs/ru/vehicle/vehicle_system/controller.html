Контроллеры — это подмодули, используемые для управления различными функциями транспортного средства с помощью lua.

Они делятся на 2 категории. Основные контроллеры, которые обрабатывают основные входные сигналы и функции транспортного средства, и вспомогательные контроллеры, позволяющие добавлять различные специальные возможности, включая электронные системы помощи водителю, дополнительные функции управления трансмиссией, различные системы приборных панелей и т.д.

Контроллеры определяются в разделе controller файла jbeam, где каждый контроллер вызывается и задаются его настройки.

В некоторых случаях, для контроллеров с большим количеством параметров, предпочтительнее использовать список вне раздела контроллера. Примеры включены в те контроллеры, где они применяются наиболее часто.

Основные

Эти контроллеры обрабатывают основные входные сигналы и функции любого транспортного средства. Каждое транспортное средство должно ссылаться хотя бы на один из них, как правило на vehicle controller

Название
Описание
Name
Vehicle Controller
Description
Основной контроллер, используемый для транспортных средств
Name
dummy
Description
Резервный контроллер, используемый если контроллер не задан. Применяется преимущественно для пропсов.
