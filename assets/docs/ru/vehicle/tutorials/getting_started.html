Необходимое программное обеспечение :

  • Программа для 3D-моделирования. Blender является наиболее распространённым выбором, поскольку он бесплатен и очень хорош. НЕ используйте САПР-программы, так как создаваемая ими геометрия далеко не оптимальна для использования в видеоиграх. Страница моделирования транспортных средств является неплохой отправной точкой. Ещё одним преимуществом Blender является наличие Blender Jbeam Editor , который может помочь вам с редактированием структур Jbeam.
  • Расширенный текстовый редактор для файлов Jbeam и Json. Visual Studio Code является рекомендуемым вариантом, поскольку режим отображения JSONC будет выделять синтаксические ошибки. Официальный плагин Jbeam станет отличным дополнением к этой программе. Альтернативой является Cursor , который имеет преимущество в виде возможности использовать ИИ для помощи с некоторыми простыми, рутинными задачами.
  • Графический редактор, например Paint.Net или GIMP , для работы с текстурами и скинами.

Всё перечисленное программное обеспечение является бесплатным.

Общие рекомендации

  • Если у вас нет опыта в моделировании, изучите общие обучающие материалы для начинающих — они не обязательно должны быть о машинах. В сети доступно огромное количество руководств для самых разных уровней знаний.

  • Начинайте с небольших проектов с малым количеством подвижных частей, так как они более снисходительны к ошибкам и проще поддаются исправлению. JBeam довольно сложен и включает множество различных компонентов, поэтому проще начинать с проектов, которые не требуют таких сложных понятий, как подвеска или комплексная деформация.

  • Попробуйте создавать JBeam с нуля в процессе обучения. Хотя в повторном использовании ванильного JBeam нет ничего плохого, вы получите гораздо более глубокое понимание, если будете создавать его самостоятельно, а не просто перемещать ноды.

  • Читайте наши руководства. Они созданы, чтобы помочь таким новичкам, как вы 😃

  • Не бойтесь просить помощи у других моддеров!
