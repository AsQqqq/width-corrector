Окраска транспортного средства — это, по сути, пресет для настроек цветовой палитры . Её можно выбрать при спауне транспортного средства через селектор машин или при перекраске с помощью меню покраски в конфигураторе.

Варианты окраски транспортного средства в селекторе машин. Они представляют заводские цвета, доступные для данного автомобиля.

Варианты окраски транспортного средства в конфигураторе. Помимо заводских окрасок, здесь также отображаются подходящие пользовательские варианты и сохранённые пресеты окрасок пользователя.

Определение окрасок

Самый простой способ задать окраску — внутри основного файла info.json транспортного средства. В этом файле каждая окраска имеет заголовок с названием и несколько базовых свойств:

  • baseColor: RGBA-цвет окраски в диапазоне 0–1 для каждого канала. Свойство Alpha определяет параметр Chrominess окраски — это устаревшая функция, не поддерживаемая системой физически корректного рендеринга (PBR).
  • clearCoat: определяет интенсивность эффекта лакового покрытия.
  • clearCoatRoughness: определяет шероховатость эффекта лакового покрытия.
  • metallic: определяет коэффициент металличности.
  • roughness: определяет коэффициент шероховатости.
"paints":[
  "Mica": {
    "baseColor": [0.5,  0.42,  0.39,  1.2],
    "clearcoat": 0,
    "clearcoatRoughness": 0,
    "metallic": 0,
    "roughness": 0.07
  },
(...)
],

Пример базового определения окраски.

Дополнительным свойством окраски является class. Если задать значение custom, окраска появится в разделе «Пользовательские» в меню покраски конфигуратора транспортного средства.

"Noce": {
  "class": "custom",
  "baseColor": [0.16,  0.1,  0.06,  1.2],
  "clearcoat": 0,
  "clearcoatRoughness": 0,
  "metallic": 0,
  "roughness": 0.07
},

Пример определения пользовательской окраски.

Окраски по умолчанию

Каждая конфигурация транспортного средства имеет собственный файл info.json с именем по схеме info_[config name].json; например, конфигурация «standard» будет иметь файл «info_standard.json». Внутри этого файла, среди прочего, можно задать название окраски по умолчанию для данной конфигурации. Именно она отображается на миниатюре и применяется при спауне, если пользователь не выбрал окраску в селекторе машин. Эта окраска по умолчанию занимает первый слот — верхний левый в селекторе окрасок.

Выбранная конфигурация транспортного средства с выделенной окраской по умолчанию в первом слоте.

Это задаётся свойством defaultPaintName1. Аналогично, defaultPaintName2 и defaultPaintName3 используются для установки окрасок по умолчанию для остальных палитр.

"defaultPaintName1": "Yellow",
"defaultPaintName2": "Yellow",
"defaultPaintName3": "Black",

Настройка окрасок по умолчанию.

Основной файл info.json транспортного средства также может содержать список окрасок по умолчанию для каждой из цветовых палитр — они применяются, если у выбранной конфигурации вовсе не определены собственные окраски. Поскольку это лишь запасной вариант, иногда задаётся только первая окраска по умолчанию, что означает использование одного и того же цвета для всех трёх палитр.

Внутренне настройка со свойствами defaultPaintName преобразуется в многоцветную схему окраски ; подробнее это будет описано на следующих страницах.

Окраски в пользовательских конфигурациях

Пользовательские конфигурации не имеют собственных файлов info.json. Настройки окраски сохраняются в файле конфигурации с расширением .pc. Настройки для всех трёх цветовых палитр сохраняются в том же формате, что и базовые окраски, но без собственных имён.

"paints":[
  {
    "baseColor":[
      0.86,
      0.66,
      0.1,
      1.2
    ],
    "clearcoat":1,
    "clearcoatRoughness":0.04,
    "metallic":0,
    "roughness":1
  },
  {
    "baseColor":[
      0.86,
      0.66,
      0.1,
      1.2
    ],
    "clearcoat":1,
    "clearcoatRoughness":0.04,
    "metallic":0,
    "roughness":1
  },
  {
    "baseColor":[
      0.83,
      0.83,
      0.83,
      1.2
    ],
    "clearcoat":1,
    "clearcoatRoughness":0.04,
    "metallic":0.3,
    "roughness":0.6
  }
],

Настройки окрасок в пользовательском файле конфигурации.
