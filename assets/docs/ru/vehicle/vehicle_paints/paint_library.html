Библиотека красок — это файл, содержащий краски и многослойные настройки красок , на которые может ссылаться любой автомобиль. Это позволяет использовать одни и те же краски для машин одного бренда, делиться универсальными пользовательскими красками и в целом упрощает процесс добавления пресетов цветов краски к автомобилю. Файл загружается до обработки автомобиля, что делает его доступным при выборе транспортного средства для спауна.

Файл библиотеки красок

Файл библиотеки красок по умолчанию хранится в vehicles/common/paintLibraries/common.paintLibrary.json. Однако любой моддер может создать файл с именем, заканчивающимся на .paintLibrary.json, и он будет работать так же, как и стандартный, добавляя новые краски и многослойные настройки красок, доступные для всех автомобилей.

Имя и/или директория пользовательского файла должны быть уникальными, например vehicles/common/myModPaints/common.paintLibrary.json или vehicles/common/paintLibraries/myMod.paintLibrary.json. Файл можно даже разместить внутри папки пользовательского автомобиля, и он всё равно будет доступен всем автомобилям — все краски из всех файлов библиотек красок, независимо от местонахождения, доступны из любого главного файла info.json любого автомобиля.

Содержимое файла библиотеки красок выглядит следующим образом:

{
  "version": 1,
  "paints": {
    "bianco_alpino": {
      "name": "Bianco Alpino",
      "baseColor": [0.85,  0.88,  0.87,  1.2],
      "clearcoat": 0,
      "clearcoatRoughness": 0,
      "metallic": 0,
      "roughness": 0.07
    },
    "blu_acciaio": {
      "name": "Blu Acciaio",
      "baseColor": [0.25,  0.3,  0.35,  1.2],
      "clearcoat": 0,
      "clearcoatRoughness": 0,
      "metallic": 0,
      "roughness": 0.07
    },
    (...)
    }
  },
  "multiPaintSetups" : {
    "team_offroad_colors" : {
      "name": "Orange Race Colors",
      "paint1": "signal_orange",
      "paint2": "palladium_gray",
      "paint3": "black_diamond"
    },
    (...)
  }
}

Определение краски внутри библиотеки красок практически идентично стандартному определению краски. Отличие состоит в том, что имя краски находится внутри списка параметров, а индекс (показан в выделенных строках) является идентификатором краски (Paint ID). Это сделано для того, чтобы в будущем можно было перевести название краски на разные языки. Параметр name краски по-прежнему используется при установке краски из библиотеки в качестве краски по умолчанию для автомобиля или конфигурации.

Название краски отображается при наведении на неё курсора мыши.

Использование красок из библиотеки

Чтобы использовать краску из библиотеки на автомобиле, добавьте простой раздел в главный файл info.json этого автомобиля:

"libraryPaints": [
  "bianco_alpino",
  "blu_costa",
  "blu_nebbia",
  "arancione",
  "blu_marina",
  "nero_puro",
  "rosso_scarlatto",
  {"id": "argento_metallizzato",       "class": "custom"},
  {"id": "arancio_metallizzato",       "class": "custom"},
  {"id": "verde_speciale_metallizzato","class": "custom"},
  {"id": "bordeaux",                   "class": "custom"},
  {"id": "rosso_fuoco",                "class": "custom"},
  {"id": "blu_tropicale",              "class": "custom"},
  {"id": "giallo_solare",              "class": "custom"},
  {"id": "giallo_savana",              "class": "custom"},
  {"id": "verde_opale",                "class": "custom"},
  {"id": "verde_edera",                "class": "custom"},
  {"id": "carbone",                    "class": "custom"},
  {"id": "blu_brillante",              "class": "custom"},
  {"id": "azzurro",                    "class": "custom"},
  {"id": "chiaro_di_luna",             "class": "custom"}
]

Обратите внимание, что данный раздел использует квадратные скобки, тогда как раздел внутри paintLibrary.json использует фигурные.

При добавлении краски из библиотеки:

  • Если указан только идентификатор, краска будет считаться заводской.
  • Если указана структура с идентификатором и параметром "class", равным "custom", краска появится во вкладке пользовательских красок.

Файл info.json, использующий как стандартные определения красок, так и ссылки на библиотеку красок, будет выглядеть следующим образом:

{
  (...)
  "defaultPaintName1": "Rosso Corallo",
  "paints": {
    "Alabastro": {
      "class": "custom",
      "baseColor": [0.84,  0.79,  0.76,  1.2],
      "clearcoat": 0,
      "clearcoatRoughness": 0,
      "metallic": 0,
      "roughness": 0.07
    },
    (...)
  },
  "libraryPaints": [
    "bianco_alpino",
    (...)
    { "id": "argento_metallizzato", "class": "custom" },
    (...)
  ]
}
