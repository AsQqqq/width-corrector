Раздел information содержит данные, которые отображаются пользователю в пользовательском интерфейсе (UI).

Раздел представляет собой простой словарь с несколькими необязательными аргументами:

string
type
Имя автора
Укажите своё имя в этом поле. Несколько авторов разделяются запятой.
BeamNG зарезервировано для официальных транспортных средств и не должно использоваться.
string
type
Отображаемое название детали

Данные в jbeam слева, отображение в игре справа:

Не может начинаться со знака $, так как он зарезервирован для выражений. Чтобы обойти это ограничение, можно поставить пробел в начале — название будет отображаться корректно, например:

"name":" $36M Pallet",
string
type
Денежная стоимость детали в долларах США
Используется в карьерном режиме для покупки и продажи деталей по реалистичным ценам.
boolean
type
false
default
Скрывать ли деталь в селекторе деталей
При значении true деталь будет отображаться в селекторе деталей только при включённой опции «Include auxiliary debug content».

Простой пример 

{
  "information":{
    "authors":"Max Mueller",
    "name":"Front Spindles",
    "value":100,
  },
}
