This feature is supported as of BeamNG.tech and .drive version 0.6.0 to latest.

Модификаторы выхлопа — это дополнительные параметры, применяемые к выхлопным нодам для изменения звука выхлопа и эффекта дожигания топлива (afterfire).

Модификатор вступает в силу, если нода является частью дерева выхлопа, соединённого с двигателем, — это позволяет создать выхлопную систему, которую можно оторвать при столкновении, отчего автомобиль начинает звучать громче.

В одном дереве может быть несколько нод с модификаторами выхлопа; их коэффициенты перемножаются. Это означает, что, например, 2 ноды с визуальным коэффициентом дожигания 0.5 на одном выхлопе дадут суммарный визуальный коэффициент дожигания 0.25.
