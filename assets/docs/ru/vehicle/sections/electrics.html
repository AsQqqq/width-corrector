This feature is supported as of BeamNG.tech and .drive version 0.34 to latest.

Electrics — это секция в Jbeam, позволяющая определять пользовательские значения electrics и назначать им сглаживатели (smoothers). Она состоит из 2 таблиц: customValues, которая определяет имя значения и функцию, из которой оно вычисляется, и smoothers, которая назначает сглаживатели значениям по имени. Пользовательские значения electrics можно использовать для плавных переходов между бинарными входами — например, для фар sealed beam, которые плавно включаются и выключаются. Их также можно применять для сложения или перемножения нескольких входных сигналов, что позволяет добиться эффекта затемнения фар при нагрузке на электрическую систему автомобиля, например при использовании стартера.

"electrics": {
    "customValues": [
        ["electricsName", "electricsFunction","electricsDefault"],
        ["customValue1","electrics.existingValue1 * electrics.existingValue2",0],
    ]
    "smoothers": [
        ["electricsName", "smootherType", "params"],
        ["customValue1","temporalNonLinear", [10,10]],
    ]
}

Вся секция electrics может существовать внутри секции components и при этом будет нормально считываться. Это удобно при глубоком слиянии таблиц — например, для добавления нового пользовательского значения или сглаживателя в другой части транспортного средства.

"components": {
    "electrics": {
        "customValues": [
            ["electricsName", "electricsFunction","electricsDefault"],
            ["customValue1","electrics.existingValue1 * electrics.existingValue2",0],
        ]
        "smoothers": [
            ["electricsName", "smootherType", "params"],
            ["customValue1","temporalNonLinear", [10,10]],
        ]
    }
}

Аргументы CustomValues

string
type
Имя нового пользовательского значения electrics.
Не может совпадать с одним из существующих значений транспортного средства. Список значений можно вывести в консоль с помощью dump(electrics) при выбранном автомобиле.
string
type
Выражение на Lua для вычисления выходного значения.

Пример выражения:

"(electrics.lowhighbeam  + electrics.brakelight_signal_L) * electrics.electricalLoadCoef"

Выражение не может содержать присваиваний или одиночных знаков «=»! Оно может содержать:

  • числа
  • все стандартные операторы Lua
  • все стандартные математические функции и константы Lua
  • имена существующих значений electrics с префиксом "electrics."
  • имена переменных, существующих на транспортном средстве, с префиксом "$"
  • разрешённые слова: "not", "true", "false", "nil", "and", "or", "dt"
  • пользовательские функции, перечисленные ниже.
number
type
Выходное значение, которое будет установлено при сбросе транспортного средства.
Если задана пустая строка, значение сохранится между сбросами (не рекомендуется). В большинстве случаев здесь используется значение 0.

Аргументы Smoothers

string
type
Имя пользовательского значения electrics, к которому будет применён сглаживатель.
string
type
Тип сглаживателя.

Поддерживаемые типы сглаживателей:

  • "exponential" — экспоненциальное сглаживание
  • "temporal" — линейное временное сглаживание
  • "temporalNonLinear" — нелинейное временное сглаживание
table
type
Параметры для выбранного сглаживателя.

Для экспоненциального сглаживания обычно используется только первый:

  • window — количество кадров для сглаживания
  • startingValue — по умолчанию 0, если не задано
  • fixedDt — шаг по времени в секундах, по умолчанию 0.0005, если не задано

Для временного сглаживания обычно используются первые два:

  • inRate — по умолчанию 1, если не задано
  • outRate — по умолчанию равен inRate, если не задано
  • autoCenterRate — существует только для линейного варианта, по умолчанию равен inRate, если не задано
  • startingValue — по умолчанию 0, если не задано

Наиболее полезный тип сглаживателя — нелинейный временной с заданными скоростями включения и выключения, используемый для переходов включения-выключения галогенных фар и фар sealed beam:

"smoothers": [
    ["electricsName", "smootherType", "params"],
    ["lowbeam_filament","temporalNonLinear", [10,10]],
    ["highbeam_filament","temporalNonLinear", [10,10]],
]

Некоторые значения electrics имеют экспоненциальный сглаживатель по умолчанию:

  • С окном в 10 кадров: wheelspeed, gear_A, rpm, lights, turnsignal, airspeed, airflowspeed, altitude
  • С окном в 100 кадров: fuel, oiltemp, watertemp

Простой пример

Пользовательские значения electrics для простых галогенных фар и фар sealed beam, которые плавно включаются и выключаются и реагируют на нагрузку электрической системы.

"electrics": {
    "customValues": [
        ["electricsName", "electricsFunction","electricsDefault"],
        ["lowbeam_filament","electrics.lowbeam * electrics.electricalLoadCoef",0],
        ["highbeam_filament","electrics.highbeam * electrics.electricalLoadCoef",0],
        ["lowhighbeam_filament","electrics.lowhighbeam * electrics.electricalLoadCoef",0],
        ["taillight_filament","electrics.lowhighbeam * electrics.electricalLoadCoef",0],
        ["brakelight_filament","electrics.brakelights * electrics.electricalLoadCoef",0],
        ["brakelight_signal_L_filament","electrics.brakelight_signal_L * electrics.electricalLoadCoef",0],
        ["brakelight_signal_R_filament","electrics.brakelight_signal_R * electrics.electricalLoadCoef",0],
        ["lowhighbeam_signal_L_filament","electrics.lowhighbeam_signal_L * electrics.electricalLoadCoef",0],
        ["lowhighbeam_signal_R_filament","electrics.lowhighbeam_signal_R * electrics.electricalLoadCoef",0],
        ["reverselight_filament","electrics.reverse * electrics.electricalLoadCoef",0],
        ["foglight_filament","electrics.fog * electrics.electricalLoadCoef",0],
        ["signal_L_filament","electrics.signal_L * electrics.electricalLoadCoef",0],
        ["signal_R_filament","electrics.signal_R * electrics.electricalLoadCoef",0],
    ]
    "smoothers": [
        ["electricsName", "smootherType", "params"],
        ["lowbeam_filament","temporalNonLinear", [10,10]],
        ["highbeam_filament","temporalNonLinear", [10,10]],
        ["lowhighbeam_filament","temporalNonLinear", [10,10]],
        ["taillight_signal_L_filament","temporalNonLinear", [15,15]],
        ["taillight_signal_R_filament","temporalNonLinear", [15,15]],
        ["taillight_filament","temporalNonLinear", [10,10]],
        ["brakelight_filament","temporalNonLinear", [10,10]],
        ["brakelight_signal_L_filament","temporalNonLinear", [15,15]],
        ["brakelight_signal_R_filament","temporalNonLinear", [15,15]],
        ["lowhighbeam_signal_L_filament","temporalNonLinear", [15,15]],
        ["lowhighbeam_signal_R_filament","temporalNonLinear", [15,15]],
        ["reverselight_filament","temporalNonLinear", [10,10]],
        ["foglight_filament","temporalNonLinear", [10,10]],
        ["signal_L_filament","temporalNonLinear", [15,15]],
        ["signal_R_filament","temporalNonLinear", [15,15]],
    ]
}

Расширенные примеры

Пользовательские значения electrics для комбинированных галогенных функциональных огней.

"electrics": {
  "customValues": [
    ["electricsName", "electricsFunction","electricsDefault"],
    ["lowhighBrakeSignal_L_filament","(electrics.lowhighbeam + electrics.brakelight_signal_L)*electrics.electricalLoadCoef",0],
    ["lowhighBrakeSignal_R_filament","(electrics.lowhighbeam + electrics.brakelight_signal_R)*electrics.electricalLoadCoef",0],
  ]
  "smoothers": [
    ["electricsName", "smootherType", "params"],
    ["lowhighBrakeSignal_L_filament","temporalNonLinear", [5,5]],
    ["lowhighBrakeSignal_R_filament","temporalNonLinear", [5,5]],
  ]
}

Пользовательское значение electrics для анимированной галогенной полицейской мигалки, которая плавно включается и выключается, постоянно мигая при включённой световой панели.

"electrics": {
    "customValues": [
        ["electricsName", "electricsFunction","electricsDefault"],
        ["police_flasher_filament","pwm(sign(electrics.lightbar), 0.5, 0.5) * electrics.electricalLoadCoef",0],
    ]
    "smoothers": [
        ["electricsName", "smootherType", "params"],
        ["police_flasher_filament","temporalNonLinear", [4,4]],
    ]
}
