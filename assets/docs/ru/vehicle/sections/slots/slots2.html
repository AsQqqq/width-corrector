This feature is supported as of BeamNG.tech and .drive version 0.31 to latest.

Slots2 — это улучшение раздела Slots, разработанное и реализованное в подготовке к новой системе слотов, запланированной для будущих обновлений игры. На данный момент оно не предоставляет явных преимуществ перед стандартным разделом Slots. Главное отличие заключается в разделении типа слота на фактическое имя слота и списки разрешённых и запрещённых slotType для данного слота. Вы можете использовать как Slots, так и Slots2 в разных Jbeam-файлах транспортного средства.

Обязательные аргументы

string
type
Внутреннее имя слота
То же понятие, что и «type» в разделе Slots.
table
type
Список slotType деталей, разрешённых в данном слоте

При прямом преобразовании из slots старое содержимое поля «type» из того раздела становится единственным элементом данной таблицы.

Позволяет устанавливать детали с несколькими вариантами slotType в один и тот же слот.

table
type
Список slotType деталей, запрещённых в данном слоте

При прямом преобразовании из slots эта таблица будет пустой.

Данный параметр имеет более высокий приоритет, чем allowTypes: если деталь содержит 2 slotType, один из которых разрешён, а другой — запрещён, она будет отклонена.

string
type
Компонент, загружаемый по умолчанию, если ни одна деталь не задана в файле конфигурации автомобиля
string
type
type
default
Название слота в селекторе деталей

Необязательные аргументы

boolean
type
Если включено, возможность оставить слот пустым удаляется из селектора деталей/гаража
dictionary
type
Содержит имена переменных в качестве ключей словаря и их значения в качестве значений словаря

Имена переменных должны начинаться с символа $. Пример использования:

["log_trailer_bolsters_R","log_trailer_bolsters_R","Rear Bolsters", {"variables": {"$bolster_offset":-0.39}}],

Часто передаваемые аргументы

Они аналогичны аргументам из раздела Slots и также поддерживают числовые суффиксы таким же образом.

dictionary
type
{"x":0, "y":0, "z":0}
default
Смещает все ноды, флексбоди и пропы по осям YZ, а также смещает ноды, флексбоди и пропы в зеркальном отражении по оси X (м)
dictionary
type
{"x":0, "y":0, "z":0}
default
Смещает все ноды, флексбоди и пропы по осям XYZ (м)
dictionary
type
{"x":0, "y":0, "z":0, "px":0, "py":0, "pz":0}
default
Поворачивает все ноды, флексбоди и пропы на x, y, z градусов по осям XYZ вокруг центральной точки px, py, pz

Простой пример

Стандартное использование slots2 без каких-либо отличий в поведении от стандартного раздела slots.

"slots2": [
    ["name", "allowTypes", "denyTypes", "default", "description"],
    ["pickup_frame", ["pickup_frame"], [], "pickup_frame", "Frame", {"coreSlot":true}],
    ["paint_design", ["paint_design"], [], "", "Paint Design"],
    ["skin_glass", ["skin_glass"], [], "", "Glass Tint"],
    ["licenseplate_design_2_1", ["licenseplate_design_2_1"], [], "", "License Plate Design"],
    ["pickup_mod", ["pickup_mod"], [], "", "Additional Modification"],
],

Расширенный пример

Использование slots2 совместно с nodeMove и переменными слота.

"slots2": [
        ["name", "allowTypes", "denyTypes", "default", "description"],
        ["simple_traffic_suspension_F",["simple_traffic_suspension_F"],[],"simple_traffic_suspension_F","Front Suspension",
        {
            "nodeMove":{"x":0.0, "y":-1.456, "z":0.00},
            "variables": {
                "$suspFUpperMountPosX":0.37,
                "$suspFUpperMountPosZ":0.57,
                "$suspFLowerMountPosX":0.37,
                "$suspFLowerMountPosZ":0.35,
            },
        }
        ],
        ["simple_traffic_suspension_R",["simple_traffic_suspension_R"],[],"simple_traffic_suspension_R","Rear Suspension",
        {
            "nodeMove":{"x":0.0, "y":1.487, "z":0.00},
            "variables": {
                "$suspRUpperMountPosX":0.43,
                "$suspRUpperMountPosZ":0.56,
                "$suspRLowerMountPosX":0.65,
                "$suspRLowerMountPosZ":0.31,
            },
        }
        ],
    ],
