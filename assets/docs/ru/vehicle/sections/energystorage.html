This feature is supported as of BeamNG.tech and .drive version 0.8.0 to latest.

Хранилище энергии используется для обозначения бака с топливом, набора аккумуляторов или баллона с закисью азота.

Обязательные аргументы

string
type
Тип хранимой энергии
Доступные варианты: "fuelTank", "n2oTank"" и "electricBattery". Каждый из них имеет уникальные параметры, описанные ниже.
string
type
Название хранилища энергии
В случае транспортного средства с несколькими топливными баками каждое название должно быть уникальным.

Топливный бак

string
type
Название балки, разрушение которой вызовет утечку из бака.
Повреждённый топливный бак будет быстро терять топливо.
string
type
Пресеты для энергетической плотности и весовой плотности.
Значения с предопределёнными энергетической и весовой плотностью: "gasoline", "diesel" и "kerosene". При использовании любого другого значения необходимо задать свойства energyDensity и fuelLiquidDensity.

Используется преимущественно при задании типа топлива, не указанного в списке. В остальных случаях использовать не рекомендуется.

Переопределение весовой плотности топлива (кг/л)
Используется преимущественно при задании типа топлива, не указанного в списке. В остальных случаях использовать не рекомендуется.

Примеры

Типичный топливный бак. Обратите внимание на переменную, используемую для начального объёма топлива

"energyStorage": [
        ["type", "name"],
        ["fuelTank", "mainTank"],
],
"mainTank": {
        "energyType":"gasoline",
        "fuelCapacity": 45,
        "startingFuelCapacity": "$fuel",
        "fuel": {"[engineGroup]:":["fuel"]},
        "breakTriggerBeam": "fuelTank",
},

Баллон закиси азота весом 10 фунтов

"energyStorage": [
        ["type", "name"],
        ["n2oTank", "mainBottle"],
],
"mainBottle": {
        "capacity": 4.54,
        "startingCapacity": 4.54,
},

Аккумулятор ёмкостью 50 кВт·ч

"energyStorage": [
        ["type", "name"],
        ["electricBattery", "mainBattery"],
],
"mainBattery": {
        "energyType":"electricEnergy",
        "batteryCapacity": 50,
        "startingCapacity": "$fuel",
},
