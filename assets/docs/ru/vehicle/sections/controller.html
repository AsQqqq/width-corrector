Контроллеры используются для симуляции различных специальных функций транспортного средства с помощью Lua. Они включают такие вещи, как цифровые приборы, режимы движения, световые балки, система стабилизации и т.д.

Каждому транспортному средству необходим как минимум один основной контроллер; основными вариантами являются "dummy" и "vehicleController" .

Стандартные контроллеры можно найти в папке "\lua\vehicle\controller" в директории установки игры.

Обязательные аргументы

string
type
Название контроллера

Дополнительные аргументы могут быть обязательными или доступными в зависимости от используемого контроллера. Подробнее см. в документации по конкретному контроллеру .

Простой пример

Определение vehicleController вместе с beamNavigator и его параметрами.

"controller": [
    ["fileName"],
    ["vehicleController", {}],
    ["beamNavigator", {"screenMaterialName": "@etk800_screen", "htmlFilePath": "local://local/vehicles/common/ETK_navi_screen.html", "name":"etk800_navi"}],
],
