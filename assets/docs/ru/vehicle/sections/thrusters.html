Траст-двигатели (thrusters) создают силу, толкающую транспортное средство вперёд. Они определяются парой нод. Сила прикладывается ко второй ноде (Node2 на изображении) и направлена в сторону первой ноды (Node1). Node1 и Node2 задаются в Jbeam как "id1:" и "id2:" соответственно.

Они используются для представления реактивных двигателей и ускорителей JATO, а также могут применяться в тестовых целях.

Тяга рассчитывается путём умножения значения "control" на коэффициент, вплоть до максимального предела, равного thrustLimit. Для electrics, значение которых изменяется от 0 до 1, коэффициент обычно совпадает с предельным значением тяги.

Чтобы активировать траст-двигатель из консоли, можно задать значению, управляющему его коэффициентом, значение 1. Например, если управляющее значение равно 'jato', команда будет выглядеть следующим образом:

electrics.values.jato = 1

Обязательные аргументы

type
Нода, определяющая направление силы
string
type
Нода, к которой прикладывается сила траст-двигателя
number
type
1
default
Множитель функции управления.
Если electrics , используемый для управления, принимает значения только до 1, это значение должно быть равно предельному значению тяги.
number
type
FLT_MAX
default
Максимальная величина тяги в ньютонах.
string
type
Параметр electrics, который будет использоваться для управления данным траст-двигателем

Подробнее см. на странице electrics .

Если установлено значение "auto", управление будет основано на текущей скорости.

Пример 

"thrusters": [
    ["id1:", "id2:", "factor", "thrustLimit", "control"],
    ["jt5l", "jt6l", 50000, 50000, "jato"],
    ["jt5r", "jt6r", 50000, 50000, "jato"],
],
