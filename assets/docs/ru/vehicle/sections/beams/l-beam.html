This feature is supported as of BeamNG.tech and .drive version 0.5.3 to latest.

L-балки — это балки, которые используются для сопротивления изменению угла между двумя балками.

Они задаются с помощью 3 нод, при этом L-балка создаётся между нодами 1 и 2. Между нодами 1 и 3, а также нодами 2 и 3 уже должны быть определены балки.

L-балки применяют усилие только при изменении угла между двумя стандартными балками, но не при изменении лишь длины самой L-балки.

Основные области применения — шины, где они используются для придания боковой жёсткости каркасу при сохранении возможности свободного сжатия боковин под нагрузкой. Также они применяются для рессор.

Обязательные аргументы

См. страницу балок .

string
type
Имя третьей ноды.

Любое изменение угла между балками [id1,id3] и [id2,id3] сопротивляется L-балкой [id1,id2].

["node1","node2", {"id3:":"node3"}],

Пример

Простая L-балка.

{"beamType":"|LBEAM"},
{"beamSpring":10000,"beamDamp":30},
{"beamPrecompression":1.0},
["node1","node2", {"id3:":"node3"}],
{"beamPrecompression":1},

Набор L-балок, используемых в качестве рессор на грузовике. Обратите внимание на высокое значение демпфирования и начальное предварительное сжатие.

//leaf spring
{"beamType":"|LBEAM"},
{"beamSpring":265140,"beamDamp":1880},
{"beamPrecompression":0.957},
["ax2r","le1r", {"id3:":"axsr","tag":"susp_RR"}],
["ax2l","le1l", {"id3:":"axsl","tag":"susp_RL"}],
["ax2r","lf1r", {"id3:":"axsr","tag":"susp_RR"}],
["ax2l","lf1l", {"id3:":"axsl","tag":"susp_RL"}],
{"beamPrecompression":1},
