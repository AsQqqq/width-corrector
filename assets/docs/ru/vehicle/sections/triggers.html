This feature is supported as of BeamNG.tech and .drive version 0.22 to latest.

Раздел Triggers используется для размещения кликабельных областей на транспортных средствах, имитирующих кнопки, переключатели, дверные ручки и другие компоненты, управляемые вручную.

Этот раздел не может существовать самостоятельно. Ему необходим раздел Event, который определяет исполняемый код, и TriggerEventLinks, связывающий их между собой.

  • events определяет фактические действия для выполнения (аналогично input_actions.json, но внутри детали)
  • triggerEventLinks связывает триггеры и events

В более новом рабочем процессе triggers2 код действий обычно определяется в файлах *.interaction.json. См. Interaction Maps и Triggers2 .

0.38.5.0 содержит 3 записи в 1 разделе, распределённые по 20043 деталям в 5090 jbeam-файлах.

Аргументы раздела TriggerEventLinks

string
type
ID триггера
string
type
Название ввода, используемого для запуска события

Часто используемые значения:

  • action0 = левая кнопка мыши
  • action1 = правая кнопка мыши
  • action2 = средняя кнопка мыши
string
type
ID события

Аргументы раздела Events

Обязательные аргументы

string
type
Уникально идентифицирует событие. Должен быть уникальным для всего транспортного средства
string
type
Заголовок действия
Очень короткое название действия, которое будет отображаться в различных меню игры
string
type
Описание действия
Полное описание, может отображаться как всплывающая подсказка; по длине должно быть около одного предложения

Необязательные аргументы

Для работы события необходимо определить хотя бы один из них.

string
type
Код, выполняемый при изменении ввода триггера
string
type
Код, выполняемый при нажатии кнопки триггера
string
type
Код, выполняемый при отпускании кнопки триггера

{

    ["id", "label", "idRef:", "idX:", "idY:","type", "size", "baseRotation", "rotation","translation", "baseTranslation"],

  ],
  "triggerEventLinks":[
    ["triggerId:triggers", "action", "targetEventId:events"],
    ["cargodoorR", "action0", "toggle_doorR"]
  ],
  "events":[
    ["id", "title", "desc"],
    ["toggle_doorR", "ui.inputActions.van.toggle_doorR.title", "ui.inputActions.van.toggle_doorR.description", {"onUp":"electrics.values.doorR = 1-(electrics.values.doorR or 0)", "order": 2}]
