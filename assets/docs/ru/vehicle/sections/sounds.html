This feature is supported as of BeamNG.tech and .drive version 0.28 to latest.

Раздел Sounds используется для переопределения, изменения или отключения определённых звуков. Он наиболее полезен для пропов — чтобы изменить звуки ударов, а также для изменения коэффициента заглушения звука внутри салона транспортного средства.

Настройка микса двигателя и выхлопа описана в разделе Настройка звука двигателя .

Необязательные аргументы

string
type
Название ноды, в которой будет воспроизводиться звук порыва ветра

Если не указано, расположение по умолчанию будет совпадать с расположением ноды звука клаксона. Если и та не задана, используется главный refNode транспортного средства.

Расположение должно быть точно выровнено по центру оси X, перед камерой капота, если только сама камера не является крайней точкой спереди транспортного средства. В идеале это должна быть жёстко закреплённая нода на кузове, близко к передней части капота. Для большинства автомобилей хорошо подходит нода в верхней средней части опоры радиатора, однако это следует подбирать индивидуально для каждого транспортного средства.

number
type
1
default
Задаёт силу фильтра звука внутри салона транспортного средства
Обычно в кузове автомобиля устанавливается значение около 0.28, которое затем увеличивается модификатором "$+cabinFilterCoef", применяемым к каждой детали, закрывающей кабину, пока в сумме не достигнет около 1.
string/boolean
type
Переопределяет стандартное звуковое событие удара транспортного средства

Использование пути к событию переопределит звук на звук, предоставляемый этим событием.

Использование значения "false" полностью отключит этот звук.

Список всех доступных путей к событиям можно найти в World Editor: Window > Audio > SFX Previewer.

number
type
1
default
Регулирует кривую громкости звука удара
При меньших значениях удары с малым усилием от тяжёлых нод будут тише, однако удары с большим усилием по-прежнему останутся громкими. Полезно при использовании высоких значений веса нод (>10kg).

Простой пример

Переопределение и отключение некоторых звуков для пропа

"sounds": {
   "impactMetal":"event:>Destruction>Props>ball_impact"
   "impactPlastic":"event:>Destruction>Props>ball_impact"
   "impactGeneric":"event:>Destruction>Props>ball_impact"
   "breakGeneric":"event:>Destruction>Props>ball_impact"
   "wind":false
   "scrapeMetal":false
   "scrapePlastic":false
},

Настройка громкости звука удара для пропа с очень большим nodeWeight (~400kg)

"sounds": {
    "impactSoundVolumeCoef":0.01,
},

Расширенный пример

Установка базового коэффициента фильтра кабины

"sounds": {
    "cabinFilterCoef": 0.28
},

Увеличение коэффициента для каждой дополнительной детали, закрывающей пассажирский отсек

"sounds": {
    "$+cabinFilterCoef": 0.03
},
