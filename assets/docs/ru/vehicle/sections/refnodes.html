Опорные ноды (ref nodes) используются для определения ориентации и положения автомобиля, а также положения камер. Они представляют собой набор из 4 нод, задающих систему координат, и 2 нод, определяющих переднюю часть транспортного средства.

Важно — refnodes задают систему координат транспортного средства для всего автомобиля.

Ноды ref:, left:, back: и up: определяют оси транспортного средства в BeamNG: +X = влево (left:), +Y = назад (back:), +Z = вверх (up:). Вперёд — это −Y, а не +X. Все позиции нод и крепления сенсоров BeamNG.tech используют эту систему отсчёта. См. Системы координат для диаграмм и заметок о мировом пространстве.

На транспортном средстве всегда должен присутствовать единственный набор refnodes, то есть он должен быть частью вашего jbeam кузова или шасси. Транспортное средство без refnodes не сможет заспавниться.

Хотя refnodes не обязаны быть точно отцентрированы на транспортном средстве, все refnodes, кроме угловых нод, должны быть идеально выровнены друг с другом в соответствующем направлении, иначе могут возникнуть проблемы с камерой и/или спавном, например транспортное средство может появиться вверх ногами. Для этого может потребоваться внести некоторые корректировки в ваш jbeam, а в отдельных случаях может понадобиться добавить ноды, служащие исключительно в качестве refNodes.

Обязательные аргументы 

"refNodes":[
    ["ref:", "back:", "left:", "up:", "leftCorner:", "rightCorner:"]
    ["fr9", "fr10", "fr9l", "fr11", "f7ll", "f7rr"],
],
