Файлы JBeam разделены на несколько секций, каждая из которых представляет один тип элемента в структуре транспортного средства.

Основные структурные элементы

Это главные элементы любого JBeam, определяющие форму и коллизии транспортного средства.

Подробнее можно прочитать во введении в JBeam

  • Системы координат - Оси транспортного средства и мира (+X влево, +Y назад, вперёд = −Y). Прочитайте это перед размещением нод или сенсоров.
  • Nodes - Первый элемент в структуре нода/балки. Они являются точечными массами и обеспечивают большинство коллизий JBeam, а также задают вес структуры.
  • Beams - Соединители между нодами. Балки обеспечивают гибкость и деформацию в структуре JBeam и определяют, как различные элементы соединены между собой.
  • Triangles - Поверхности, определяемые между нодами. Треугольники используются для расчётов аэродинамики и для коллизий между транспортными средствами.

Продвинутые структурные элементы

Эти элементы позволяют создавать более сложные структурные конструкции и могут быть очень полезны в конкретных применениях.

  • Hydros - Балки, длину которых можно изменять по команде.
  • Rails and slidenodes - Позволяет определять ноды, скользящие вдоль рейки. Используется для таких элементов, как рулевые рейки, жёсткость некоторых конструкций и т.д.
  • Thrusters - Позволяет непосредственно приводить конструкции в движение. Используется для тестирования, JATO и других целей.
  • Torsion bars - Создаёт крутильное сопротивление между двумя рычагами. Используется для стабилизаторов поперечной устойчивости и для жёсткости некоторых конструкций.
  • Torsion hydros - Аналогично hydros, но изменяет угол между двумя рычагами. Используется в некоторых рулевых системах и других типах актуаторов.

Система деталей и слотов

Эти элементы в основном используются в рамках системы деталей/слотов — либо для определения дополнительной информации, связанной с деталью, либо для описания слотов внутри детали.

  • Information - Определяет несколько основных параметров, связанных с компонентом, например отображаемое имя в селекторе деталей.
  • Slots - Определяет список слотов внутри компонента, а также выбор по умолчанию.

Графические и звуковые элементы

Эти секции обрабатывают различные дополнительные элементы транспортного средства.

  • Flexbodies - Определяет модели, которые будут использоваться для графического представления вашего транспортного средства, и способ их привязки к нодам для деформации.
  • Glowmaps - Позволяет определять материалы, которые могут переключаться по требованию. В основном используется для смены материала фар на эмиссионный при их включении.
  • Soundscape - Определяет звуковые эффекты, воспроизводимые при выполнении определённых условий, а также ноду, используемую в качестве источника звука.
  • Cameras - Определяет положение и угол различных камер в транспортном средстве, включая внешние и внутренние камеры.
  • Props - Анимированные 3D-элементы, которые не деформируются вместе со структурой JBeam. Используются для таких объектов, как приборы, педали, рулевые колёса и т.д.

Трансмиссия

Последняя секция содержит элементы, связанные с компонентами трансмиссии, такими как определения силового агрегата и колёса.

  • Powertrain - Используется для определения различных элементов трансмиссии в составе силового агрегата транспортного средства.
  • Pressure Wheels - Позволяет определять колёса с моделью шины с давлением.
  • Rotators - Альтернатива колёсам с давлением в специализированных применениях, где мощность должна передаваться на что-то иное, нежели колесо, например на пропеллер.

Прочее

  • Variables - Позволяет определять регулируемые параметры транспортного средства, как правило регулируемые компоненты подвески и трансмиссии.
  • Triggers - Определяет заданные триггеры, которые игрок может активировать мышью для управления различными функциями транспортного средства.
  • Refnodes - Набор нод, определяющих центральную систему координат транспортного средства (см. Системы координат для обозначений направлений осей)
  • Controller - Используется для добавления специальных функций к транспортному средству, например мигалок полицейских машин, электронных систем помощи и т.д.
