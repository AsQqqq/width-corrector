Чтобы создать новую миссию, войдите в режим Freeroam на нужном уровне (карте), где вы хотите её создать. Начните с открытия редактора с помощью клавиши F11, затем перейдите в Window -> Missions -> Mission Editor

В редакторе миссий слева отображается список миссий. Вы можете фильтровать их с помощью зелёного значка фильтра рядом с полем поиска. Рекомендуется установить флажок «Current Level Only» и задать группировку по «type».

Чтобы создать миссию, выберите File -> New mission.

  • Name : Название миссии, отображаемое игроку
  • ID : Внутреннее имя миссии. Оно должно быть уникальным и предпочтительно содержать ваш никнейм или иной уникальный идентификатор, чтобы избежать дублирования ID при публикации в репозитории. Первая часть ID (до первого слэша) обязательно должна быть идентификатором уровня (карты), иначе миссия может не отображаться на карте.
  • AutoID : Автоматически генерирует новый ID на основе карты и названия миссии
  • MissionType : Тип миссии. Основной выбор — между Flowgraph и различными заранее заданными шаблонами.

Все эти параметры, кроме «ID», можно изменить после создания миссии.

После заполнения всех полей нажмите «Create Mission».

Общая информация

Название и описание — это информация, которую игрок видит перед началом миссии.

Стартовый триггер

Стартовый триггер определяет положение начальной точки миссии.

Его можно перемещать, как и любой другой объект редактора карты. Некоторые полезные сочетания клавиш: Shift + — поместить начальную точку в место клика; Shift + — изменить размер начальной точки.

Дополнительная информация

VehicleUsed носит информативный характер и сообщает игроку, какой транспорт он может использовать в миссии (собственный, предоставленный и т. д.).

Difficulty отображается на экране миссии и даёт игроку примерное представление о её сложности.

Превью

Preview позволяет добавить изображение, которое будет показано на экране миссии. В папке миссии должен находиться файл с именем «preview.png», «preview.jpg» или «preview.jpeg» — к ней можно перейти с помощью кнопки «…».

После того как вы скопируете изображение, нажмите «reload-all» — красный восклицательный знак должен смениться зелёной галочкой. Если навести курсор на кнопку «preview», можно увидеть предпросмотр изображения.

Условия запуска

Условия запуска — это требования, которые должны быть выполнены, прежде чем игрок сможет начать миссию.

Условия можно удалять с помощью значка корзины, а добавлять — кнопкой «add». Количество условий не ограничено.

  • BusDriven : Проверяет, что игрок управляет автобусом
  • vehicleDriven : Проверяет, что игрок управляет транспортным средством

Типы миссий

Типы миссий делятся на 2 основные категории: готовые шаблоны миссий и flowgraph.

Большинство типов в списке — это заранее заданные шаблоны, позволяющие создавать простые миссии: доехать до зоны, дрифт, прыжок в длину, точная парковка и т. д. Все они поддерживают определённый уровень настройки: стартовая точка, используемый транспорт, ограничение по времени и т. д. Подробнее см. страницу заранее заданных типов миссий .

Тип flowgraph позволяет дополнительно настраивать миссии с помощью редактора графического программирования. Он несколько сложнее готовых шаблонов, но значительно гибче. Подробнее см. страницу миссий на основе flowgraph .
