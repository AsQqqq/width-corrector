В этом разделе описывается, как заполнять растительностью большие территории и управлять ею, сохраняя читаемость и производительность.

4.1 Основные концепции

  • Система Forest размещает растительность с учётом плотности, случайности и фильтров окружающей среды
  • Ассеты растительности требуют правильно выставленных точек вращения (пивотов), масштаба и, при необходимости, упрощённых коллизий
  • Импосторы/билборды используются для отображения объектов на большом расстоянии там, где это доступно

4.2 Требования и соглашения

  • Ассеты растительности следует создавать в реальном масштабе, с пивотами у логического основания объекта (например, у корней дерева)
  • Коллизии должны быть лёгкими там, где они необходимы, и отсутствовать для мелких декоративных объектов там, где это безопасно
  • Количество видов и вариантов следует ограничивать, чтобы контролировать количество вызовов отрисовки и потребление памяти

4.3 Состав биома

  • Организуйте растительность по логическим слоям (полог леса, подлесок, напочвенный покров)
  • Используйте ограничения по углу склона и высоте для правдоподобного размещения видов
  • Поддерживайте ограниченную палитру, чтобы избежать визуального шума
  • Инструмент Biome может помочь моддеру настроить всё необходимое

Интерфейс инструментов Forest tool и Biome tool

4.4 LOD и импосторы

  • Если моддер хочет создать собственный элемент растительности, для сложных мешей необходимо предоставить LOD-уровни. Используйте импосторы или дальние представления там, где это возможно, и ознакомьтесь с тем, как настроить управление ветром, чтобы сделать сцену более правдоподобной

Взаимодействие листвы с ветром настраивается через вертекс-пейнтинг в Blender непосредственно на меше. Пользователь может изменять только параметры Wind в уже доступных ассетах.

Настройка ветра на пользовательских мешах листвы — продвинутая функция, которую нельзя изменить внутри World Editor. Для этого необходимо стороннее программное обеспечение для создания контента, например Blender.

  • Настройте дистанции затухания, чтобы минимизировать появление объектов на скорости движения

Настройка ветра листвы с помощью инструмента shape tool

4.5 Настройка инструмента Forest tool

  • Первый шаг — добавить меши, которые моддер хочет использовать в качестве кистей (Brushes). Меши представляют собой список для выбора и настройки параметров каждого отдельного элемента леса

Список мешей в инструменте Forest tool

  • После настройки списка мешей моддер может перейти на вкладку Brushes и добавить одиночные или групповые кисти, которыми можно управлять через выбор при размещении
  • Выберите одну или несколько кистей, чтобы одновременно размещать больше объектов леса, и управляйте количеством размещаемых объектов с помощью ползунков Size (размер кисти) и Density (плотность)

Вкладка Brushes в инструменте Forest tool

4.6 Рекомендации по производительности

  • Ограничьте количество уникальных видов и материалов в пределах одного обзора
  • Отдавайте предпочтение атласам текстур для мелких растений и декоративных объектов, чтобы сократить количество вызовов отрисовки
  • Соблюдайте баланс между плотностью и читаемостью; оставляйте линии обзора вдоль дорог и у ключевых видовых точек

4.7 Расположение данных

  • Меши растительности: /YourMap/art/shapes/trees — для элементов, специфичных для карты
  • Управляемый список элементов леса сохраняется в managedItemData.json: /YourMap/art/forest
  • Данные размещения отдельных элементов сохраняются в виде файлов *.forest4.json в /YourMap/forest

4.8 Типичные проблемы

  • Слишком густые участки, вызывающие падение частоты кадров
  • Парящие или частично закопанные ассеты из-за неправильно выставленных пивотов или изменений рельефа
  • Паттерны повторяемости, заметные с возвышенных точек обзора
