Этот раздел посвящён дорожным системам и проезжим поверхностям, их материалам, топологии и интеграции с рельефом.

3.1 Основные концепции

  • DecalRoad проецирует дорожные покрытия, разметку, края дороги, дорожные детали и все основные элементы, которые могут располагаться поверх поверхности рельефа. Подходит для большинства дорог.
  • Декали обеспечивают высокочастотные детали, такие как разметка, грязь, трещины и заплатки.
  • MeshRoad генерирует геометрию; подходит для мостов, эстакад и сложных поперечных сечений.
  • Поведение покрытия определяется groundmodel-моделями через назначение материалов. Доступные groundmodel-модели можно просмотреть через верхнее меню: Window > Debug > GroundModel Debug.

Окно отладки GroundModel Debug

3.2 Основные требования и соглашения

  • Decalroads
    • размещаются нажатием ALT+LMB на поверхности для расстановки узлов сплайна decalroad

Размещение DecalRoad с помощью узлов сплайна

  • Decals
    • размещаются через редактор Decal путём выбора материала из списка Templates

Редактор декалей с шаблонами материалов

  • Соблюдайте согласованную ширину полос, радиусы кривизны и поперечный уклон, соответствующие типу дороги. Инструмент дорожного сплайна поможет создать очень детализированную и функциональную основу дорожного покрытия и должным образом подогнать рельеф под неё
  • Используйте согласованные материалы для дороги и обочины и убедитесь, что их groundmodel-модели соответствуют предполагаемому поведению при езде
  • Держите количество узлов сплайна разумным; добавляйте подразделения только там, где необходим дополнительный контроль
  • Управляйте компланарными перекрытиями с помощью параметра renderPriority в окне Inspector. Это предотвращает мерцание на перекрёстках и переходах
  • Используйте опцию overObject для размещения decalroad поверх статических мешей

DecalRoad, размещённый поверх статического меша с использованием overObject

3.3 Перекрёстки и переходы

  • Выравнивайте сегменты точно, чтобы избежать зазоров
  • Используйте декали для разметки, заплаток и стыков
  • Плавно вписывайте края дороги в прилегающие материалы рельефа (например, гравийные обочины)

3.4 Особенности MeshRoad

  • Материалы верхней, боковой и нижней сторон meshroad можно задать в инспекторе
  • Убедитесь, что коллизия непрерывна по всем пролётам (мосты, поднятые сегменты)

Верхний и боковой материалы, назначенные meshroad через окно Inspector

3.5 Рекомендации по производительности

  • Отдавайте предпочтение повторному использованию материалов для дорог и разметки.
  • Держите количество декалей разумным; используйте атласы декалей там, где это возможно, чтобы сократить количество переключений состояний.
  • Минимизируйте лишние узлы сплайна; поддерживайте плавную кривизну для снижения сложности редактирования

3.6 Расположение данных

  • Материалы дорог и декали: /art/roads и /art/decals. Ссылки указаны в main.materials.json

3.7 Типичные проблемы

  • Z-конфликт на перекрёстках из-за наложенных друг на друга декалей и decalroads с одинаковым renderPriority
  • Резкие изменения ширины или поперечного уклона, влияющие на управляемость
  • Несогласованное назначение материалов между соединёнными сегментами
