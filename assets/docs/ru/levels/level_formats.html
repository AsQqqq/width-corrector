В этом разделе описаны все основные файлы и форматы, используемые в уровнях.

Используйте этот раздел для ознакомления с компоновкой файлов и форматами сериализованных данных, специфичными для уровней: items.level.json, файлы рельефа, файлы префабов, данные леса, декали, навигационные карты, маршруты автобусов, объекты инфраструктуры, дрифт-споты, участки трасс (crawls), полосы для дрэга, площадки, сигналы, датаблоки и метаданные.

Общий обзор рабочего процесса — как связаны папки геймплея уровня, трафик, миссии, граф навигации, миникарта и данные большой карты — см. в разделе Level Gameplay Data .

Информацию о файлах групп транспортных средств, используемых трафиком, парковками, миссиями и инструментами редактора, см. в разделе Vehicle Groups .

Описание поведения объектов и полей, специфичных для классов, см. в разделе Level Object Classes .
